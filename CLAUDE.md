# CLAUDE.md

## リポジトリ概要

chezmoi で管理された dotfiles リポジトリ。

## chezmoi ルール

このリポジトリのファイルは chezmoi によってホームディレクトリに展開される。
ファイルの追加・変更時は以下の手順を守ること。

### 新規ファイル追加

1. まずターゲットパス（例: `~/.config/nvim/lua/plugins/foo.lua`）にファイルを作成する
2. `chezmoi add <ターゲットパス>` で chezmoi の管理対象に追加する
3. ソースディレクトリ（このリポジトリ）に自動的に正しい名前で配置される

**ソースディレクトリに直接ファイルを作成してはいけない。** chezmoi はソースに直接置いたファイルを管理対象として認識しない。

### 既存ファイルの編集

ソースディレクトリ内のファイルを直接編集してよい。編集後 `chezmoi apply` で反映する。

### 確認コマンド

- `chezmoi managed` — 管理対象ファイル一覧
- `chezmoi diff` — ソースとターゲットの差分確認
- `chezmoi apply --dry-run` — 適用前のプレビュー

### ディレクトリ命名規則

chezmoi はソースディレクトリで特殊なプレフィックスを使う：
- `dot_` → `.`（例: `dot_zshrc` → `.zshrc`）
- `private_` → パーミッション 0600
- `encrypted_` → age で暗号化されたファイル
